Shared expense management platform designed to replace a spreadsheet-based flatmate expense tracker.

#Tech Stack
Frontend
- React
- React Query
- Tailwind CSS
- Axios

Backend
- Node.js
- Express
- PostgreSQL
- Prisma
- JWT

Deployment
- Vercel
- Render
- Neon PostgreSQL

#Features
- Authentication
- Group management
- Membership timelines
- Expense tracking
- Equal split calculations
- Balance calculation
- Debt simplification
- Settlement recording
- CSV import
- Anomaly detection
- Import review workflow

#Setup
#Backend

cd backend

npm install

Create .env

DATABASE_URL=...
JWT_SECRET=...

npm run dev

#Frontend

cd clientside

npm install

Create .env

VITE_API_URL=http://localhost:5000/api

npm run dev
